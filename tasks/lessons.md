> Format : problème original → cause → solution → règle à retenir.

---

## Format d'entrée

```
### [DATE] — Titre court du problème

**Problème original** : Ce qui s'est passé / le symptôme observé.

**Cause** : Pourquoi c'est arrivé.

**Solution** : Ce qui a été fait pour corriger.

**Règle à retenir** : La règle concrète pour éviter de répéter cette erreur.
```
